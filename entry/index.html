応募要項

URL : 関西春ロボコン2024　応募要項

関東春ロボコンに出場するチームは関東春ロボコンHPを参照の上応募してください

出場チーム紹介

富山大学 / 富-1

富山大学ロボコンプロジェクトに所属している１年生で構成されており、先輩な どの有識者の方にサポートしていただきながら、機体の製作、制御を行いました。 使ったことのない機構もありますが、頑張ります。

富山大学 / Richesse Montague

私たちのチームは、確実性のあるロボットを作ることを目標に製作してきました。頼りになるチームメンバーと共にロボットを作ることができました。当日も全力を尽くせるよう頑張ります！

三重大学 / M³RC

私たちは部員8人で3年生もいないながら、ほとんど独学で日々試行錯誤し頑張れるチームです！ 前回の大会では、決勝トーナメントで勝ち上がることが出来なかったので、今年こそは優勝できるように頑張ります！

立命館大学 / アポロ3号

やる気のある人や、ロボット製作が好きな人、技術力のある人など、いろんな特技や個性を持った人が協力し合えるチームです。大会では優勝を目指します。

奈良工業高等専門学校 / からくり部

私たちは1年生と2年生で構成されたチームです。最速でひなまつりを達成するために機構はシンプルな動作のものにし、全自動で動くようにしました。チームの目標である優勝を目指して頑張ります。

京都大学 / 京大機械研究会

京大機械研究会の1回生で構成されたチームです。走行機能のあるロボットを作るのは初めてなので、試行錯誤しながら開発しています。本番でいい結果を出せるよう頑張っていきます。

京都工芸繊維大学 / 動け!パペットマン

【急募】アットホームな職場です。安定した動きができるパペットマンを目指し、日々設計加工制御をして暮らしています。雛壇を完成させひなまつりを楽しめるよう頑張ります。

同志社大学 / メカメカモンスターズ

我々のロボットは昨年の反省点を踏まえ、足回りの簡素化や雛人形と台座の回収機構の共通化等によりメンテナンス性の向上を図りました。大会では存分にパフォーマンスが発揮できるよう頑張ります！

舞鶴工業高等専門学校 / さんしょくやまき

ソフト面で色々な問題が発生してしまいましたがそれらを乗り越えて，去年と同じように良い戦績を残せるよう頑張ります！

舞鶴工業高等専門学校 / ひな舞鶴

一年生のみで構成されていて全員が初めてなのもあり、意見のすれ違いが発生することもありました。それでもなお立ち向かい続けた私たちの姿、今まで積み上げてきたこの努力の結晶をぜひ見てください。

大阪大学 / Robohan

私たち「Robohan」は24時間365日ロボコンに熱中し、研鑽を重ねています。後期の単位全てを捧げ、春ロボの準備をしてきました。柵越え機構を搭載したロボット「Amembo」で春ロボ優勝を目指します。

大阪公立大学工業高等専門学校 / ねやがわ☆ロボコンｽﾞ

メンバーの2/3以上がとよはし☆ロボコンズ進学者のチームです。シンプルかつ誤差を吸収できる機構とROS2による自動制御で、高専ロボコンで"やり残した"優勝とロボットの自動化を目指します！

大阪公立大学工業高等専門学校 / 卍HIN-G卍

私たち卍HIN-G卍は、2・3年生の高学年チームに、助っ人として1年生を1人加えたチームです。相手が自動ロボットでも必ず勝てる「秘策」を駆使して優勝を目指します！！

大阪公立大学工業高等専門学校 / 卍曼荼羅卍

私たちは、ロボットの設計から制作までを全て１年生が担当しました。個性豊かな機構とロボットを作ることができました。１年間ロボット倶楽部で学んだ技術などを全て発揮したいです。

神戸市立工業高等専門学校 / 神戸高専 Aチーム

神戸高専Aチームは一年生だけのチームです。全員初めてのロボット制作なのでまだまだ未熟な点がありますが、一年生ならではの斬新な動きに注目してください。

神戸市立工業高等専門学校 / 神戸高専 Bチーム

無駄のない設計による、ロボットの安定感がアピールポイントです。最速で雛壇を完成させるために、精錬された動きを可能にしました。部員初めての自動制御の挑戦、成功させられるように、頑張ります！

明石工業高等専門学校 / 明石高専 Aチーム

明石高専Aチームは、とにかく速くひなまつりを達成することを目標にロボットを開発しました。無駄のない回収機構、素早い足回り、木枠の乗り越え機構。技術とロマンを詰め込んだロボットで、競技優勝を目指します!

明石工業高等専門学校 / 明石高専 Bチーム

僕たち明石高専Bチームは1年生だけで構成されたチームです。アピールポイントに同学年しかいないので強い結束力があることです。先輩方には技術力では敵わないですが精一杯自分たちの実力を出し切りたいです。

明石工業高等専門学校 / 明石高専 Cチーム

ロボコンに初めて本格的に参加する1年生だけでのチームです。 経験が少なく、春ロボ参加にあたって様々な困難がありましたが チーム一丸となって乗り越えてきました。 本番でのぼんぼり点灯を目指して頑張ります！！！

香川高等専門学校(高松キャンパス) / ひなどり

私たちひなどりはひなまつり達成を目標にロボットを制作しました。まとめて2つの台座を回収し、シンプルな機構で雛人形を回収、配置します。大会でひなまつりを達成できるよう頑張ります。

九州大学 / United States of Koala

私たちは安心、安全を意識してロボットを制作してきました。機体制作の遅れや機体規模の縮小が生じたり、全自動化を断念したりしましたが試合では確実に動作するロボットをお見せできればと思っています。

熊本高等専門学校(八代キャンパス) / 八代ロボコン部

チーム「八代ロボコン部」のロボットは、機械を1,2年生で作っており、1つの機構で台座と雛人形の両方を運ぶことができます。また、Vゴール狙いで自動制御を採用しています。初参加ですが、優勝目指して頑張ります。
